# Base Variables
Variables that can be used across a wide range of commands and invoke messages. 

## Guild Variables
Guild variables can be useful in areas such as welcome messages when displaying your overall member count & more.

| `{guild.count}`                   | Member count                   | `150`                                                |
| `{guild.region}`                  | Server region                  | `us-west`                                            |
| `{guild.shard}`                   | Shard ID                       | `0`                                                  |
| `{guild.owner_id}`                | Server owner ID                | `123456789012345678`                                 |
| `{guild.created_at}`              | Server creation timestamp      | `1672574400`                                         |
| `{guild.emoji_count}`             | Number of emojis in the server | `20`                                                 |
| `{guild.role_count}`              | Number of roles in the server  | `10`                                                 |
| `{guild.boost_count}`             | Number of boosts               | `5`                                                  |
| `{guild.boost_tier}`              | Boost level                    | `2`                                                  |
| `{guild.preferred_locale}`        | Preferred language             | `en-US`                                              |
| `{guild.key_features}`            | Key features of the server     | `VIP_REGIONS, ANIMATED_ICON`                         |
| `{guild.icon}`                    | Server icon URL                | `https://cdn.discordapp.com/icons/.../icon.png`      |

## Channel Variables
Variables to view information about the channel where the script is sent to.

| Variable                   |  Description              | Example                  |
| -------------------------- | ------------------------- | ------------------------ |
| `{channel}`                | Channel mention           | `#general`               |
| `{channel.name}`           | Channel name              | `general`                |
| `{channel.id}`             | Channel ID                | `123456789012345678`     |
| `{channel.mention}`        | Channel mention           | `<#123456789012345678>`  |
| `{channel.type}`           | Channel type              | `text`                   |
| `{channel.position}`       | Channel's position        | `1`                      |
| `{channel.topic}`          | Channel topic             | `Welcome to the server!` |
| `{channel.category_id}`    | Category ID               | `234567890123456789`     |
| `{channel.category_name}`  | Category name             | `Information`            |
| `{channel.slowmode_delay}` | Slowmode delay in seconds | `5`                      |


## Lastfm Variables
Lastfm variables are useful in areas where you want to display both track and artist info on your `;nowplaying` embed.
<Tabs>
<Tab title="Track">
| Variable                  | Output                                     | Example                                                           |
|---------------------------|--------------------------------------------|-------------------------------------------------------------------|
| `{track.name}`           | Shows the track's name                    | `30 For 30 (with Kendrick Lamar)`                                 |
| `{track.image}`          | Shows the track's image                   | `https://lastfm.freetls.fastly.net/i/u/300x300/nums.jpg`          |
| `{track.plays}`          | Shows the track's play count              | `16`                                                             |
| `{track.url}`            | Shows the track's URL                     | `https://www.last.fm/music/SZA/_/30+For+30+(with+Kendrick+Lamar)`                        |
  </Tab>

  <Tab title="Artist">
| Variable                  | Output                                     | Example                                                           |
|---------------------------|--------------------------------------------|-------------------------------------------------------------------|
| `{artist.name}`           | Shows the artist's name                    | `Kendrick Lamar`                                                  |
| `{artist.image}`          | Shows the artist's image                   | `https://lastfm.freetls.fastly.net/i/u/300x300/nums.jpg`          |
| `{artist.plays}`          | Shows the artist's play count              | `609`                                                             |
| `{artist.url}`            | Shows the artist's URL                     | `https://www.last.fm/music/Kendrick+Lamar`                        |
  </Tab>

</Tabs>
